# Captcha Detector Pro

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)  
[![Version](https://img.shields.io/badge/version-1.0-blue.svg)](https://github.com/HabboLens)

**Captcha Detector Pro** is a lightweight notification tool for **Habbo Agency virtual environments**. It monitors your screen for CAPTCHAs using preloaded templates and alerts the user with a beep whenever a CAPTCHA appears, so you can respond manually.

## Features
- Detects Habbo Agency CAPTCHAs on-screen using preloaded templates  
- Alerts the user with a beep sound  
- Fast, lightweight, minimal system resource usage  
- Simple and intuitive interface  
- No automatic solving; manual action required
